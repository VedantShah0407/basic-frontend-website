About This Project

This project was built as a learning exercise to understand the core concepts of front-end web development. It uses only HTML5 and CSS, without any external libraries or frameworks.

The goal of this website is to demonstrate:

  • Semantic HTML5 structure
  • Simple and clean UI design
  • Multi-page navigation
  • Responsive layout for different screen sizes

Projects like this help build a strong foundation before moving on to advanced technologies like JavaScript frameworks and backend development.
