# Basic Front-End Website

A visually attractive static website built using HTML and CSS, focusing on clean layout,
modern color combinations, and user-friendly design.

## Features
- Clean and structured page layout
- Responsive design fundamentals
- Simple and intuitive user interface

## Technologies Used
- HTML5
- CSS3

## Project Purpose
This project was created to practice front-end development fundamentals and improve
understanding of website structure, styling, and basic responsive design.
